## Conversie Tracking Setup

### 1. Google Ads Conversietag plaatsen
Plaats deze code in `<head>` van index.html (al toegevoegd in deze repo):
```html
<script>
function gtag_report_conversion(url) {
  var callback = function() {
    if (typeof(url) != 'undefined') { window.location = url; }
  };
  gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXXX/CONVERSION_LABEL',
    'value': 90.0,
    'currency': 'EUR',
    'event_callback': callback
  });
  return false;
}
</script>
```

### 2. Conversie-acties aanmaken in Google Ads
1. Ga naar Tools & Settings → Conversions
2. Klik "Nieuwe conversie-actie"
3. Kies "Website"
4. Voer domein in: `salderingsdienst.nl`
5. Maak deze conversies aan:

| Conversie | Trigger | Waarde | Count |
|-----------|---------|--------|-------|
| Boeking afspraak | Bevestigingspagina boeking | €90 | Één |
| Lead formulier | Start boeking (stap 1) | €20 | Één |
| Telefoontje | Klik op WhatsApp/telefoon | €15 | Één |
| Paginabezoek | Kennisbank artikel (>2min) | €5 | Één |

### 3. Enhanced Conversions
Activeer Enhanced Conversions voor nauwkeuriger meting:
- Ga naar Conversion Settings → Enhanced Conversions
- Schakel in voor e-mailadres (hashed)

## Optimalisatie Schema

### Week 1–2: Opstart
- Alle campagnes op "Maximaal aantal klikken"
- Dagbudget per campagne halveren (€55 totaal)
- Daily check: zoektermenrapport → negative keywords toevoegen
- Quality Score monitoren

### Week 3–4: Leren
- Campagnes met <5% CTR pauzeren of herschrijven
- Beste headlines behouden, slechtste vervangen
- Biedstrategie wijzigen naar "Maximaal aantal conversies"

### Maand 2: Schalen
- Budget verhogen naar €110/dag als CPA < €40
- Beste advertentiegroepen extra budget geven
- Display remarketing opzetten (bezoekers website)

### Maand 3+: Optimaliseren
- Doel-CPA instellen op €30–35
- A/B testen met nieuwe headlines
- Seizoensafhankelijk aanpassen (meer budget in dec/jan)
